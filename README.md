# Call of Duty Weapon and Camo Tracker

## Project Overview
This Qt-based desktop application allows users to manually track their progress for weapons and camouflages in Call of Duty. The intuitive interface lets players input and update their unlock progress for each weapon, including camo challenges and achievements.

## Features
- **User-Friendly Interface**: Easily navigate through weapons and camouflages.
- **Progress Tracking**: Input and update unlock progress for each weapon.
- **Filtering Options**: Filter weapons by type for better organization.
- **Data Storage**: Save and load progress from .json.

## Usage
- Open the application.
- Select a weapon from the list.
- Update the progress for camouflages and achievements.
- Save your progress to track your in-game goals effectively.
